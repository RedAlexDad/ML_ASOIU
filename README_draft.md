**Функция Розенброка:**
```
f(x) = Σ[i=1 to n-1] [150(x_i² - x_{i+1})² + 2(x_i - 1)²] + 100
```

**Ограничения:**
```
g1(x) = x1² + x2² - x3 - 1 ≤ 0
g2(x) = -x1 ≤ 0  (x1 ≥ 0)
g3(x) = -x2 ≤ 0  (x2 ≥ 0)
g4(x) = -x3 ≤ 0  (x3 ≥ 0)
```
