## Методы многомерного поиска

**Вариант 2**

| Параметр | Значение |
|----------|----------|
| a | 150 |
| b | 2 |
| f₀ | 100 |
| n | 3 |

**Функция Розенброка:**
```
f(x) = Σ[i=1 to n-1] [150(x_i² - x_{i+1})² + 2(x_i - 1)²] + 100
```

### Пример использования API

```python
from optimization import Optimizer, create_rosenbrock_function, create_visualization
import numpy as np

# Создание функции с параметрами
f, grad_f, _ = create_rosenbrock_function(a=150, b=2, f0=100)

# Начальная точка
x0 = np.array([0.0, 0.0, 0.0])

# Создание оптимизатора
optimizer = Optimizer(f, grad_f, x0)

# Запуск всех методов
results = optimizer.run_all_methods()

# Сравнение методов
optimizer.print_comparison_table()

# Получение лучшего метода
best_name, best_result = optimizer.get_best_method('time')
print(f"Лучший метод: {best_name}")
```
