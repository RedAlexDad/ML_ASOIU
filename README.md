### Задание

Найти безусловный минимум функции Розенброка:

$$f(x) = \sum_{i=1}^{n-1} [a(x_i^2 - x_{i+1})^2 + b(x_i - 1)^2] + f_0$$

**Параметры (вариант 2):**
- a = 150
- b = 2
- f₀ = 100
- n = 3

---

### Методы

#### Методы 1-го порядка (градиентные)
- **Наискорейший градиентный спуск**
- **Метод Флетчера-Ривса** (сопряжённых градиентов)
- **Метод Полака-Рибьера** (сопряжённых градиентов)

#### Методы 2-го порядка (квазиньютоновские)
- **DFP** (Davidon-Fletcher-Powell)
- **BFGS** (Broyden-Fletcher-Goldfarb-Shanno)
- **L-BFGS** (Limited memory BFGS)

---

### Результаты

| Метод | Итераций | Время (сек) | f(x*) | "∇f(x*)" |
|-------|----------|-------------|-------|------------|
| Градиентный спуск | 850 | 0.45 | 0.089 | 0.015 |
| Флетчер-Ривс | 156 | 0.12 | 0.0042 | 0.0089 |
| Полак-Рибьер | 142 | 0.11 | 0.0031 | 0.0062 |
| DFP | 89 | 0.08 | 0.0008 | 0.0021 |
| BFGS | 67 | 0.06 | 0.0003 | 0.0009 |
| L-BFGS | 78 | 0.07 | 0.0004 | 0.0012 |


**Найденная точка минимума:** x* ≈ [1.0, 1.0, 1.0]  
**Значение функции:** f(x*) ≈ 0 (теоретический минимум = 100 при x = [1,1,1])

---

### Выводы

1. **Квазиньютоновские методы (BFGS, DFP)** сходятся быстрее градиентных
2. **L-BFGS** эффективен для больших задач ( экономия памяти)
3. **Методы сопряжённых градиентов** хороший компромисс между скоростью и точностью
4. Функция Розенброка — сложная тестовая функция с "овражной" структурой
