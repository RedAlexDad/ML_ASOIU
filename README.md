### Задание

Найти минимум функции Розенброка с ограничениями:

$$f(x) = \sum_{i=1}^{n-1} [a(x_i^2 - x_{i+1})^2 + b(x_i - 1)^2] + f_0$$

**Параметры (вариант 2):**
- a = 150, b = 2, f₀ = 100, n = 3

**Ограничения:**
- g₁(x) = x₁² + x₂² - x₃ - 1 ≤ 0
- g₂(x) = -x₁ ≤ 0 (x₁ ≥ 0)
- g₃(x) = -x₂ ≤ 0 (x₂ ≥ 0)
- g₄(x) = -x₃ ≤ 0 (x₃ ≥ 0)
