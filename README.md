### ✅ LR1: Обработка признаков (часть 1)

**Тема:** Пропуски, кодирование, нормализация

**Задание:**
1. Устранение пропусков в данных
2. Кодирование категориальных признаков
3. Нормализация числовых признаков

**Датасет:** Продажи электромобилей (1200 записей, 22 признака)

**Методы:**
- Пропуски: среднее, медиана, мода, KNN-импутация
- Кодирование: Label Encoding, One-Hot Encoding
- Нормализация: StandardScaler, MinMaxScaler

**Файл:** `main.ipynb`

---

### ✅ LR2: Обработка признаков (часть 2)

**Тема:** Масштабирование, выбросы, отбор признаков

**Задание:**
1. Масштабирование признаков (≥3 способа)
2. Обработка выбросов (удаление + замена)
3. Обработка нестандартного признака
4. Отбор признаков (Filter, Wrapper, Embedded)

**Методы:**
- Масштабирование: StandardScaler, MinMaxScaler, RobustScaler, MaxAbsScaler
- Выбросы: IQR, Isolation Forest
- Отбор: SelectKBest, RFE, Lasso

**Файл:** `main.ipynb`

---

### ✅ LR3: Методы многомерного поиска

**Тема:** Оптимизация функции Розенброка

**Задание:** Найти минимум функции Розенброка методами 1-го и 2-го порядка

**Параметры:** a=150, b=2, f₀=100, n=3

**Методы:**
| Метод | Итераций | f(x*) |
|-------|----------|-------|
| Градиентный спуск | 850 | 0.089 |
| Флетчер-Ривс | 156 | 0.004 |
| Полак-Рибьер | 142 | 0.003 |
| DFP | 89 | 0.0008 |
| BFGS | 67 | 0.0003 |
| L-BFGS | 78 | 0.0004 |

**Лучший:** BFGS (67 итераций, f(x*) ≈ 0)

---

### ✅ LR4: Условная оптимизация

**Тема:** Методы поиска условного экстремума

**Задание:** Минимум функции Розенброка с ограничениями

**Ограничения:**
- g₁(x) = x₁² + x₂² - x₃ - 1 ≤ 0
- g₂(x) = -x₁ ≤ 0, g₃(x) = -x₂ ≤ 0, g₄(x) = -x₃ ≤ 0

**Методы:**
| Метод | Итераций | f(x*) | Допустимо |
|-------|----------|-------|-----------|
| Штрафные функции | 1 | 100.00 | ✗ |
| Барьерные функции | 8 | 119.75 | ✓ |
| Комбинированный | 1 | 100.06 | ✓ |
| **Множители Лагранжа** | 100 | **100.00** | ✓ |
| Проекция градиента | 1000 | 100.02 | ✓ |

**Лучший:** Метод множителей Лагранжа
