## Методические указания

Даны дважды непрерывно дифференцируемые целевая функция $f(x) = f(x_1, x_2, \dots, x_n)$ и функции ограничений:

$$g_j(x) = 0, \quad j = 1, \dots, l; \qquad g_k(x) \le 0, \quad k = l+1, \dots, m$$

определяющие множества допустимых решений $X$.

Требуется найти локальный минимум целевой функции на множестве $X$, т.е. такую точку $x^* \in X$, что:

$$f(x^*) = \min_{x \in X} f(x) \tag{4.1}$$

где:

$$X = \left\{ x \ \middle|\ \begin{aligned} &g_j(x) = 0, \quad j = 1, \dots, l; \quad l < n \\ &g_k(x) \le 0, \quad k = l+1, \dots, m \end{aligned} \right\}$$

Применение необходимых и достаточных условий, использованных в Лабораторной Работе № 2, эффективно для решения ограниченного числа примеров, в которых вытекающие из этих условий соотношения имеют аналитическое решение. Для решения большинства практических задач используются численные методы, которые делятся на две группы.

### Группа 1

**Методы, использующие преобразование задачи условной оптимизации в последовательность задач безусловной оптимизации** путем введения в рассмотрение вспомогательных функций: **методы последовательной безусловной минимизации**.

### Группа 2

**Методы непосредственного решения задачи условной оптимизации**, основанные на движении из одной допустимой точки, где выполнены все ограничения, к другой допустимой точке с лучшим значением целевой функции. Эти методы часто называют **методами возможных направлений**.

Основная идея методов первой группы состоит в том, чтобы аппроксимировать исходную задачу условной оптимизации некоторой вспомогательной задачей, решение которой менее сложно, чем решение исходной. Естественно, что ограничившись решением одной лишь вспомогательной задачи, можно получить, вообще говоря, лишь приближенное решение. Если же использовать последовательность задач, в определенном смысле «сходящихся» к исходной, то искомое точное решение окажется пределом соответствующей последовательности приближенных решений. Идея преобразования задачи с ограничениями в определенным образом построенную последовательность задач без ограничений представляется заманчивой главным образом в связи с наличием эффективных алгоритмов безусловной минимизации, рассмотренных в Лабораторной Работе № 3.

На практике для получения решения исходной задачи с требуемой точностью бывает достаточно решить конечное (относительно небольшое) число вспомогательных задач. При этом нет необходимости решать их точно, а информацию, полученную в результате решения очередной вспомогательной задачи, обычно удается эффективно использовать для решения следующей.

В рамках единой методологии можно выделить несколько подходов к решению задачи:

1. **Первый подход** называется **методом штрафов (внешних штрафов)**. В этом методе к целевой функции добавляется функция, интерпретируемая как штраф за нарушение каждого из ограничений. Метод генерирует последовательность точек, которая сходится к решению исходной задачи.

2. **Второй подход** называется **методом барьеров (внутренних штрафов)**. В этом случае к целевой функции исходной задачи добавляется слагаемое, которое не позволяет генерируемым точкам выходить за пределы допустимой области.

3. **Третий подход** связан с добавлением штрафной функции не к целевой функции, а к функции Лагранжа. В результате возникает модифицированная функция Лагранжа, а методы, использующие эту функцию, называются **методом множителей Лагранжа**.

4. **Четвертый подход** базируется на введении так называемых **точных штрафных функций**, позволяющих ограничиться решением лишь одной задачи безусловной минимизации.

К методам второй группы относятся методы непосредственного решения задачи условной оптимизации. Они связаны с нахождением предела $x^*$ последовательности $\{x^k\}$ допустимых точек при $k \to \infty$, таких, что $f(x^{k+1}) < f(x^k)$, $k = 0, 1, \dots$.

Последовательность $\{x^k\}$ строится по правилу $x^{k+1} = x^k + \delta x^k$, $k = 0, 1, \dots$, где вектор $\delta x^k$ определяется в зависимости от применяемого метода. К этой группе относятся метод проекции градиента и метод возможных направлений Зойтендейка.

## 4.1. Методы последовательной безусловной минимизации

### 4.1.1. Метод штрафных функций (метод внешних штрафов)

#### Стратегия поиска

Идея метода заключается в сведении задачи на условный минимум к решению последовательности задач поиска безусловного минимума вспомогательной функции:

$$F(x, r^k) = f(x) + P(x, r^k) \to \min_{x \in \mathbb{R}^n}$$

где $P(x, r^k)$ – штрафная функция, $r^k$ – параметр штрафа, задаваемый на каждой $k$-й итерации. Это связано с возможностью применения эффективных и надежных методов поиска безусловного экстремума, рассмотренных в Лаб.Раб. № 3.

Штрафные функции конструируются, исходя из условий:

$$P(x, r^k) = \begin{cases} 
0, & \text{при выполнении ограничений} \\
> 0, & \text{при невыполнении ограничений}
\end{cases}$$

Причем при невыполнении ограничений и $r^k \to \infty$, $k \to \infty$ справедливо $P(x, r^k) \to \infty$.

Чем больше $r^k$, тем больше штраф за невыполнение ограничений. Как правило, для ограничений типа равенств используется квадратичный штраф, а для ограничений типа неравенств – **квадрат срезки** (т.е. используется функция Хевисайда):

$$H(g_j(x)) = \begin{cases} 
0, & \text{при } g_j(x) \le 0 \\
1, & \text{при } g_j(x) > 0
\end{cases}$$

где $g_j^+(x)$ – срезка функции:

$$g_j^+(x) = \max\{0, g_j(x)\} = \begin{cases} 
g_j(x), & \text{при } g_j(x) > 0 \\
0, & \text{при } g_j(x) \le 0
\end{cases}$$

Штрафная функция имеет вид:

$$P(x, r^k) = \frac{r^k}{2} \left\{ \sum_{j=1}^{l} [g_j(x)]^2 + \sum_{j=l+1}^{m} [g_j^+(x)]^2 \right\}$$

**Начальная точка поиска задается обычно вне множества допустимых решений** $X$. На каждой $k$-й итерации ищется точка $x^*(r^k)$ минимума вспомогательной функции $F(x, r^k)$ при заданном параметре $r^k$ с помощью одного из методов безусловной оптимизации. Полученная точка $x^*(r^k)$ используется в качестве начальной на следующей итерации, выполняемой при возрастающем значении параметра штрафа. При неограниченном возрастании $r^k$ последовательность точек $x^*(r^k)$ стремится к точке условного минимума $x^e$.

#### Алгоритм

**Ш.1.** Задать начальную точку $x^0$; начальное значение параметра штрафа $r^0 > 0$; число $C < 1$ для увеличения параметра ($C \in [4, 10]$); малое число $\varepsilon > 0$ для остановки алгоритма. Положить $k = 0$.

**Ш.2.** Составить вспомогательную функцию:

$$F(x, r^k) = f(x) + \frac{r^k}{2} \left\{ \sum_{j=1}^{l} [g_j(x)]^2 + \sum_{j=l+1}^{m} [g_j^+(x)]^2 \right\}$$

**Ш.3.** Найти точку $x^*(r^k)$ безусловного минимума функции $F(x, r^k)$ по $x$ с помощью какого-либо метода (нулевого, первого или второго порядка):

$$F(x^*(r^*), r^k) = \min_{x \in \mathbb{R}^n} F(x, r^k)$$

При этом задать все требуемые выбранным методом параметры. В качестве начальной точки взять $x^k$. Вычислить $P(x^*(r^k), r^k)$.

**Ш.4.** Проверить условие окончания:
- **а)** Если $P(x^*(r^*), r^k) \le \varepsilon$, процесс поиска закончить:
  $$x^* = x^*(r^k), \quad f(x^*) = f(x^*(r^k))$$
- **б)** Если $P(x^*(r^*), r^k) > \varepsilon$, положить:
  $$r^{k+1} = C r^k, \quad x^{k+1} = x^*(r^k), \quad k = k + 1$$
  Перейти на Ш.2.

#### Сходимость

1. Точки $x^*(r^k)$ в алгоритме – это точки локального минимума функции $F(x, r^k)$. Однако функция $F(x, r^k)$ может быть неограниченной снизу и процедуры методов безусловной минимизации могут расходиться. Это обстоятельство необходимо учитывать при программной реализации.

2. В методах штрафных функций имеется тесная связь между значениями параметра штрафа и множителями Лагранжа для регулярной точки минимума:
   $$\lambda_j(r^k) = r^k g_j[x^*(r^k)], \quad j = 1, \dots, l$$
   $$\mu_j(r^k) = r^k g_j^+[x^*(r^k)], \quad j \in J_a$$
   $$\lim_{r^k \to \infty} \lambda_j(r^k) = \lambda_j^*, \quad j = 1, \dots, l; \qquad \lim_{r^k \to \infty} \mu_j(r^k) = \mu_j^*, \quad j \in J_a$$

3. Обычно выбирается $r^0 = 1.0$, а $C \in [4, 10]$.

4. При решении задач процедура расчетов завершается при некотором конечном значении параметра штрафа $r^k$. При этом приближенное решение, как правило, не лежит в множестве допустимых решений, т.е. ограничения задачи не выполняются. Это является одним из недостатков метода. С ростом параметра штрафа $r^k$ генерируемые алгоритмом точки приближаются к решению исходной задачи извне множества допустимых решений.

5. На практике для получения решения исходной задачи с требуемой точностью достаточно бывает решить конечное (относительно небольшое) число вспомогательных задач. При этом нет необходимости решать их точно, а информацию, полученную в результате решения очередной вспомогательной задачи, обычно удается использовать для решения следующей.

### 4.1.2. Метод барьерных функций (метод внутренних штрафов)

#### Стратегия поиска

Идея метода заключается в сведении задачи на условный минимум к решению последовательности задач поиска безусловного минимума вспомогательной функции:

$$F(x, r^k) = f(x) + P(x, r^k) \to \min_{x \in X}$$

где $P(x, r^k)$ – штрафная функция, $r^k$ – параметр штрафа.

Как правило, используются:

**а) Обратная штрафная функция:**
$$P(x, r^k) = -r^k \sum_{j=1}^{m} \frac{1}{g_j(x)}$$

**б) Логарифмическая штрафная функция:**
$$P(x, r^k) = -r^k \sum_{j=1}^{m} \ln[-g_j(x)]$$

Обе штрафные функции определены и непрерывны внутри множества $X$, т.е. на множестве $X^\circ = \{x \mid g_j(x) < 0, j = 1, \dots, m\}$, и стремятся к бесконечности при приближении к границе множества изнутри. Поэтому они называются барьерными функциями.

Логарифмическая штрафная функция положительна при $-1 < g(x) < 0$ и отрицательна при $g(x) < -1$, т.е. внутренним точкам отдается предпочтение перед граничными точками.

Начальная точка задается только внутри множества $X$. На каждой $k$-й итерации ищется точка $x^*(r^k)$ минимума вспомогательной функции $F(x, r^k)$ при заданном параметре $r^k$ с помощью одного из методов безусловной минимизации. Полученная точка $x^*(r^k)$ используется в качестве начальной на следующей итерации, выполняемой при уменьшающемся значении параметра штрафа. При $r^k \to +0$ последовательность точек $x^*(r^k)$ стремится к точке условного минимума $x^*$. Барьерные функции как бы препятствуют выходу из множества $X$, а если решение задачи лежит на границе, то процедура метода приводит к движению изнутри области к границе.

#### Алгоритм

**Ш.1.** Задать начальную точку $x^0$ внутри области $X$; начальное значение параметра штрафа $r^0 > 0$; число $C > 1$ для уменьшения параметра штрафа ($C = 10$); малое число $\varepsilon > 0$ для остановки алгоритма. Положить $k = 0$.

**Ш.2.** Составить вспомогательную функцию:

$$F(x, r^k) = f(x) - r^k \sum_{j=1}^{m} \frac{1}{g_j(x)} \quad \text{или} \quad F(x, r^k) = f(x) - r^k \sum_{j=1}^{m} \ln[-g_j(x)]$$

**Ш.3.** Найти точку $x^*(r^k)$ минимума функции $F(x, r^k)$ с помощью какого-либо метода (нулевого, первого или второго порядка) поиска безусловного минимума с проверкой принадлежности текущей точки внутренности множества $X$. При этом задать все требуемые выбранным методом параметры. В качестве начальной точки взять $x^k$. Вычислить:

$$P(x, r^k) = -r^k \sum_{j=1}^{m} \frac{1}{g_j(x)} \quad \text{или} \quad P(x, r^k) = -r^k \sum_{j=1}^{m} \ln[-g_j(x)]$$

**Ш.4.** Проверить выполнение условия окончания:
- **а)** Если $|P(x^*(r^k), r^k)| \le \varepsilon$, процесс поиска закончить:
  $$x^* = x^*(r^k), \quad f(x^*) = f(x^*(r^k))$$
- **б)** Если $|P(x^*(r^k), r^k)| > \varepsilon$, положить:
  $$r^{k+1} = \frac{r^k}{C}, \quad x^{k+1} = x^*(r^k), \quad k = k + 1$$
  Перейти на Ш.2.

#### Замечание 4.1

1. Обычно выбирается $r^0 = 1$, а параметр $C = 10$.

2. При $r^k \to +0$ обеспечивается сходимость, однако с уменьшением $r^k$ функция $F(x, r^k)$ становится все более овражной. Поэтому полагать $r^k$ малым нецелесообразно.

3. Так как большинство методов поиска безусловного экстремума использует дискретные шаги, то вблизи границы шаг может привести в точку недопустимой области. Если в алгоритме отсутствует проверка на принадлежность точки множеству $\{x \mid g_j(x) < 0, j = 1, \dots, m\}$, то это может привести к ложному успеху, т.е. уменьшению вспомогательной функции в точке, где она теоретически не определена. Поэтому на шаге 3 алгоритма требуется явная проверка того, что точка не покинула допустимую область. Процедура поиска обычно завершается при некотором малом $r^k$, отличном от нуля.

4. Побочным продуктом вычислений в методе штрафных функций является вектор множителей Лагранжа:
   - **Для обратной штрафной функции:**
     $$\lambda_j(r^k) = \frac{r^k}{g_j^2(x^*(r^k))}, \quad j = 1, \dots, m$$
   - **Для логарифмической штрафной функции:**
     $$\lambda_j(r^k) = -\frac{r^k}{g_j(x^*(r^k))}, \quad j = 1, \dots, m$$
   - **Предельное значение:**
     $$\lambda_j^* = \lim_{r^k \to +0} \lambda_j(r^k)$$

### 4.1.3. Метод модифицированных функций Лагранжа

#### Стратегия поиска

Стратегия аналогична используемой в методе внешних штрафов, только штрафная функция добавляется не к целевой функции, а к классической функции Лагранжа. В результате задача на условный минимум сводится к решению последовательности задач поиска безусловного минимума модифицированной функции Лагранжа:

$$L(x, \lambda^k, \mu^k, r^k) = f(x) + \sum_{j=1}^{l} \lambda_j^k g_j(x) + \frac{r^k}{2} \sum_{j=1}^{l} [g_j(x)]^2 + \frac{1}{2r^k} \sum_{j=l+1}^{m} \left\{ \left[\max(0, \mu_j^k + r^k g_j(x))\right]^2 - (\mu_j^k)^2 \right\}$$

где:
- $\lambda^k = (\lambda_1^k, \dots, \lambda_l^k)^T$
- $\mu^k = (\mu_{l+1}^k, \dots, \mu_m^k)^T$ – векторы множителей Лагранжа
- $r^k$ – параметр штрафа
- $k$ – номер итерации

Задается начальная точка поиска $x^0$. На каждой $k$-й итерации ищется точка минимума модифицированной функции Лагранжа при заданных $\lambda^k, \mu^k, r^k$ с помощью одного из методов безусловной минимизации. Полученная точка $x^*(\lambda^k, \mu^k, r^k)$ используется в качестве начальной на следующей итерации, выполняемой при возрастающем значении параметра штрафа $r^k$ и пересчитанных определенным образом векторах множителей $\lambda^k, \mu^k$. Для достижения сходимости в отличие от метода внешних штрафов не требуется устремлять $r^k$ к бесконечности.

#### Алгоритм

**Ш.1.** Задать начальную точку $x^0$; начальное значение параметра штрафа $r^0 > 0$; число $C > 1$ для увеличения параметра штрафа ($C \in [4, 10]$); начальные значения векторов множителей $\lambda^0, \mu^0$; малое число $\varepsilon > 0$ для остановки алгоритма. Положить $k = 0$.

**Ш.2.** Составить модифицированную функцию Лагранжа:

$$L(x, \lambda^k, \mu^k, r^k) = f(x) + \sum_{j=1}^{l} \lambda_j^k g_j(x) + \frac{r^k}{2} \sum_{j=1}^{l} [g_j(x)]^2 + \frac{1}{2r^k} \sum_{j=l+1}^{m} \left\{ \left[\max(0, \mu_j^k + r^k g_j(x))\right]^2 - (\mu_j^k)^2 \right\}$$

**Ш.3.** Найти точку $x^*(\lambda^k, \mu^k, r^k)$ безусловного минимума функции по $x$ с помощью какого-либо метода (нулевого, первого или второго порядка):

$$L(x^*, \lambda^k, \mu^k, r^k) = \min_{x \in \mathbb{R}^n} L(x, \lambda^k, \mu^k, r^k)$$

При этом задать все требуемые выбранным методом параметры. В качестве начальной точки взять $x^k$.

**Ш.4.** Вычислить $P(x^*(\lambda^k, \mu^k, r^k), \mu^k, r^k)$, где:

$$P(x, \mu^k, r^k) = \sum_{j=1}^{l} [g_j(x)]^2 + \frac{1}{2r^k} \sum_{j=l+1}^{m} \left\{ \left[\max(0, \mu_j^k + r^k g_j(x))\right]^2 - (\mu_j^k)^2 \right\}$$

и проверить выполнение условия окончания:
- **а)** Если $P(x^*(\lambda^k, \mu^k, r^k), \mu^k, r^k) \le \varepsilon$, процесс поиска заканчивается:
  $$x^* = x^*(\lambda^k, \mu^k, r^k), \quad f(x^*) = f(x^*(\lambda^k, \mu^k, r^k))$$
- **б)** Если $P(x^*(\lambda^k, \mu^k, r^k), \mu^k, r^k) > \varepsilon$, положить:
  - $r^{k+1} = C r^k$ (пересчет параметра штрафа)
  - $\lambda^{k+1} = \lambda^k + r^k g(x^*(\lambda^k, \mu^k, r^k))$ (пересчет множителей для ограничений типа равенств)
  - $\mu_j^{k+1} = \max\{0, \mu_j^k + r^k g_j(x^*(\lambda^k, \mu^k, r^k))\}$ (пересчет множителей для ограничений типа неравенств)
  - $x^{k+1} = x^*(\lambda^k, \mu^k, r^k)$, $k = k + 1$
  
  Перейти на Ш.2.

#### Замечание 4.2

1. На каждой итерации желательно, чтобы найденная точка локального минимума $x^*(\lambda^k, \mu^k, r^k)$ была бы ближайшей к $x^e$. Метод корректен, если начиная с некоторого $k$ метод безусловной минимизации всякий раз приводил к одной и той же точке $x^e$ локального минимума.

Иногда выбирают $\lambda^0 = \mu^0 = 0$. В этом случае первая вспомогательная задача минимизации совпадает с решаемой в методе внешних штрафов.

При этом для достижения сходимости не требуется устремлять $r^k$ к бесконечности. Это приводит к тому, что проблема увеличения овражности не является такой острой, как в методе штрафов.

## 4.2. Методы возможных направлений

### 4.2.1. Метод проекции градиента

#### Постановка задачи

Найти минимум дифференцируемой функции $f(x) = f(x_1, \dots, x_n)$ при ограничениях $g_j(x) \le 0, j = 1, \dots, m$, т.е. такую точку $x^* \in X$, что:

$$f(x^*) = \min_{x \in X} f(x)$$

$$X = \{x \mid g_j(x) \le 0, j = 1, \dots, m\}$$

где функции $g_j(x) \le 0, j = 1, \dots, m$, являются дифференцируемыми функциями $x$.

#### Стратегия поиска

Стратегия поиска решения задачи учитывает тот факт, что решение $x^*$ может лежать как внутри, так и на границе множества допустимых решений.

Для определения приближенного решения $x^*$ строится последовательность точек $\{x^k\}$:

$$x^{k+1} = x^k + \delta x^k, \quad k = 1, \dots$$

где приращение $\delta x^k$ определяется в каждой точке $x^k$ в зависимости от того, где ведется поиск – внутри или на границе множества допустимых решений.

Решение задачи начинается с обхода границы допустимой области. Обход границы множества допустимых решений связан с выявлением активных в точке $x^k$ ограничений $g_A = (g_1, \dots, g_p)^T$, аппроксимацией их плоскостью:

$$A_k \delta x = \tau_k \tag{5.2.1}$$

где:
- $A_k = \left[\dfrac{\partial g_j}{\partial x_i}\right]_{x=x_k}$ – матрица размера $p \times n$, $p \le n$
- $\tau_k = -g_A(x^k)$

и проекцией на неё вектора $-\alpha^k \nabla f(x^k)$. Для выявления неравенств, активных в точке $x^k$, задается погрешность определения активных ограничений $\varepsilon_1 \le 0$. Активными считаются те ограничения, для которых $\varepsilon_1 \le g_j(x) \le 0$. Число $p$ ограничений, активных в точке $x^k$, не должно превышать $n$ – размерности вектора $x$.

Поиск ограничений, активных в точке $x^k$, рассматривается как самостоятельная задача, которая может быть решена путем последовательных приближений. Задается точка $x^k$ и вычисляется $g_j(x), j = 1, \dots, m$. Если $g_j(x) < \varepsilon$, то выбираются любые $p$ ограничений с наименьшими по абсолютной величине невязками $\rho_j^k = g_j(x^k)$ в точке $x^k$, строится матрица $A_k$, вычисляется $\tau_k$ и находится точка:

$$x^{k+1} = x^k + (A_k)^T (A_k (A_k)^T)^{-1} \tau_k \tag{5.2.2}$$

затем снова вычисляются невязки выбранных $p$ ограничений. Уточнение по формуле (5.2.1) осуществляется до тех пор, пока не будет найдена точка $x^k$, в которой $\varepsilon_1 \le g_j(x) \le 0$. Проекция вектора $-\alpha^k \nabla f(x^k)$ в точке $x^k$, в которой активны $p$ ограничений, определяется точкой:

$$x^{k+1} = x^k - \alpha^k \left[E - (A_k)^T (A_k (A_k)^T)^{-1} A_k\right] \nabla f(x^k) \tag{5.2.3}$$

где приращение $\delta x^k = -\alpha^k \left[E - (A_k)^T (A_k (A_k)^T)^{-1} A_k\right] \nabla f(x^k)$ осуществляет движение по плоскости $A_k \delta x = \tau_k$ в направлении убывания $f(x^k)$. Величина $\alpha^k$ выбирается так:

$$\alpha^k = \min[\alpha^{*k} \ge 0, \alpha_{\min}^k \ge 0]$$

где $\alpha^{*k}$ есть шаг, при котором:

$$f(x^k - \alpha^{*k} [E - (A_k)^T (A_k (A_k)^T)^{-1} A_k] \nabla f(x^k)) = \min_{\alpha^k} f(x^k - \alpha^k [E - (A_k)^T (A_k (A_k)^T)^{-1} A_k] \nabla f(x^k))$$

а $\alpha_{\min}^k$ – наименьший шаг, при котором $g_j(x^k - \alpha^k [E - (A_k)^T (A_k (A_k)^T)^{-1} A_k] \nabla f(x^k)) \le 0$ для всех ограничений, которые не были активными в точке $x^k$. Разумеется, невязка ограничений в точке $x^{k+1}$ изменяется, и поэтому вычисление точки $x^{k+2}$ должна предшествовать процедура выбора активных ограничений, описанная выше.

Процедура вычисления точек последовательности $\{x^k\}$ обеспечивает последовательность движения вдоль границы допустимой области. При выполнении неравенства $\|\Delta x^k\| = \|-[E - (A_k)^T (A_k (A_k)^T)^{-1} A_k] \nabla f(x^k)\| \le \varepsilon_2$, где $\varepsilon_2$ заданное достаточно малое положительное число, вычисляется приближение $\lambda^k$ вектора множителей Лагранжа $\lambda^*$:

$$\lambda^k = -(A_k (A_k)^T)^{-1} A_k \nabla f(x^k) \tag{5.2.4}$$

Если $\lambda^k \ge 0$, то в точке $x^k$ выполнены необходимые условия минимума и в ней должны быть проверены достаточные условия. Если среди множителей $\lambda_j^k$ есть отрицательные, то это означает, что $x^k$ не является приближением точки $x^*$, так как в ней не выполняются условия минимума $f(x)$ при ограничениях $g_j(x) \le 0, j = 1, \dots, m$. Однако выбор шага $\alpha^k$ позволяет говорить о том, что значение $f(x)$ не может быть уменьшено при заданном составе активных ограничений и, следовательно, процесс минимизации $f(x)$ необходимо продолжить, уменьшив их число: в число пассивных ограничений переводят то из ограничений, которому соответствует наибольший по абсолютной величине отрицательный множитель $\lambda_j^k$. Такая процедура поиска позволяет отыскать решение, лежащее как на границе, так и внутри множества допустимых решений.

#### Алгоритм

**Ш.1.** Задать начальную точку $x^0$; $\varepsilon_1 \le 0$, $\varepsilon_2 > 0$, число итераций $M$.

**Ш.2.** Положить $k = 0$.

**Ш.3.** Проверить выполнение условия $k \ge M$:
- **а)** Если неравенство выполнено, то расчет окончен, перейти к вычислению $\lambda^k$ и оценке результата;
- **б)** Иначе перейти на Ш.4.

**Ш.4.** Вычислить $g_j(x^k), j = 1, \dots, m$.

**Ш.5.** Проверить выполнение условий $\varepsilon_1 \le g_j(x^k) \le 0, j = 1, \dots, m$:
- **а)** Если неравенство выполнено хотя бы для одного $j$, вычислить $\nabla f(x^k)$.
  - Если $\nabla f(x^k) \ne 0$, перейти на Ш.7.
  - Если $\nabla f(x^k) = 0$ при $k > 0$, перейти на Ш.9, а если $\nabla f(x^0) = 0$, то следует проверить точку $x^0$ на принадлежность области допустимых решений. Если $x^0 \in X$, перейти на Ш.9. В противном случае задать новую точку $x^0$ и перейти на Ш.4.
- **б)** Если ни одно из условий не выполнено, то перейти на Ш.6.

**Ш.6.** Вычислить точку $x^\nu$, в которой будет выполнено условие $\varepsilon_1 \le g_j(x^\nu) \le 0$, по крайней мере для одного значения $j$:

$$x^\nu = x^k + (A_k)^T (A_k (A_k)^T)^{-1} \tau_k, \quad \text{где } \tau_k = -g_A(x^k)$$

Положить $x^k = x^\nu$ и перейти на Ш.7.

**Ш.7.** Вычислить $\Delta x^k = -[E - (A_k)^T (A_k (A_k)^T)^{-1} A_k] \nabla f(x^k)$.

**Ш.8.** Проверить условие $\|\Delta x^k\| \le \varepsilon_2$:
- **а)** Если неравенство выполняется, перейти на Ш.9.
- **б)** Если нет, то перейти на Ш.10.

**Ш.9.** Вычислить вектор $\lambda^k = -(A_k (A_k)^T)^{-1} A_k \nabla f(x^k)$. Если $\lambda^k \le 0$, то расчет окончен, проверить достаточные условия минимума. Если нет, то исключить из состава активных ограничений (оно переводится в пассивные), которому соответствует наибольший по модулю отрицательный множитель, и перейти на Ш.7 (при этом из матрицы $A_k$ удаляется строка, соответствующая исключенному ограничению).

**Ш.10.** Получить точку $x^k + \alpha^k \Delta x^k$.

**Ш.11.** Определить $\alpha^k$. Для этого следует:
- **а)** Вычислить $\alpha^k$ из условия $f(x^k + \alpha^k \Delta x^k) \to \min_{\alpha^k \ge 0}$;
- **б)** Для всех пассивных в точке $x^k$ ограничений, кроме переведенных в пассивные на Ш.9, определить величину $\alpha_j^k$ из условий $g_j(x^k + \alpha^k \Delta x^k) = 0, \alpha^k \ge 0$ (если условие $g_j(x^k + \alpha^k \Delta x^k) = 0$ выполняется только при $\alpha^k < 0$, то $\alpha_j^k$ не вычисляется);
- **в)** Найти величину $\alpha_{\max}^k = \min_j \{\alpha_j^k\}$;
- **г)** Вычислить значение $\alpha^k = \min\{\alpha^{*k}, \alpha_{\max}^k\}$.

**Ш.12.** Вычислить $x^{k+1} = x^k + \alpha^k \Delta x^k$. Положить $k = k + 1$ и перейти на Ш.3.
