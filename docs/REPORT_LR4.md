# Отчёт по лабораторной работе №4

## Методы поиска условного экстремума

---

### 1. Постановка задачи

**Вариант 2:** a = 150, b = 2, f₀ = 100, n = 3

**Целевая функция** (функция Розенброка):
$$f(x) = \sum_{i=1}^{n-1} [a(x_i^2 - x_{i+1})^2 + b(x_i - 1)^2] + f_0$$

**Ограничения:**
- $g_1(x) = x_1^2 + x_2^2 - x_3 - 1 \leq 0$ — нелинейное ограничение
- $g_2(x) = -x_1 \leq 0$ — ограничение неотрицательности ($x_1 \geq 0$)
- $g_3(x) = -x_2 \leq 0$ — ограничение неотрицательности ($x_2 \geq 0$)
- $g_4(x) = -x_3 \leq 0$ — ограничение неотрицательности ($x_3 \geq 0$)

**Начальная точка:** $x_0 = (0.5, 0.5, 0.5)$

---

### 2. Аналитическая проверка

**Точка (1, 1, 1):**
- $f(1,1,1) = 100.0$
- $g_1(1,1,1) = 1 + 1 - 1 - 1 = 0 \leq 0$ ✓
- $g_2(1,1,1) = -1 \leq 0$ ✓
- $g_3(1,1,1) = -1 \leq 0$ ✓
- $g_4(1,1,1) = -1 \leq 0$ ✓

**Точка (1,1,1) является допустимой и достигает теоретического минимума f = 100.**

### 4. Анализ результатов

#### 4.1. Метод штрафных функций
- **Найденная точка:** $x^* = (1.00000405, 1.0000094, 1.00001848)$
- **Значение функции:** $f(x^*) = 100.0$ — оптимальное значение
- **Итераций:** 1 — минимальное число
- **Время:** 0.0045 с — очень быстрый
- **Проблема:** небольшое нарушение ограничения $g_1(x) = 0.000008 > 0$
- **Вывод:** Метод быстро находит хорошее приближение, но не гарантирует допустимость

#### 4.2. Метод барьерных функций
- **Найденная точка:** $x^* = (0.55, 0.55, 0.55)$
- **Значение функции:** $f(x^*) = 119.19$ — значительно выше оптимума
- **Итераций:** 8
- **Время:** 0.0065 с
- **Преимущество:** 100% допустимость всех ограничений
- **Недостаток:** застревание вдали от оптимума
- **Вывод:** консервативный метод, гарантирует допустимость, но с потерей точности

#### 4.3. Комбинированный метод
- **Найденная точка:** $x^* = (0.92548426, 0.85928729, 0.7295129)$

- **Итераций:** 1
- **Время:** 0.0033 с — самый быстрый
- **Преимущество:** допустимость + близость к оптимуму
- **Вывод:** лучший баланс между скоростью, точностью и допустимостью

#### 4.4. Метод множителей Лагранжа

### 6. Выводы

1. **Для данной задачи оптимальное решение:** $x^* = (1, 1, 1)$ с $f(x^*) = 100$

2. **Метод штрафных функций** — самый быстрый и точный по значению функции, но нарушает ограничения. Рекомендуется для задач, где небольшие отклонения допустимы.

3. **Комбинированный метод** — лучший выбор для практического применения: минимальное время, допустимое решение, близкое к оптимуму.

4. **Метод барьерных функций** — слишком консервативен, застревает вдали от оптимума.

5. **Метод множителей Лагранжа** — точен, но вычислительно затратен.

6. **Метод проекции градиента** — неэффективен для функции Розенброка из-за её "овражной" структуры.

7. **Рекомендация:** для задач условной оптимизации с функцией Розенброка рекомендуется использовать **комбинированный метод** или **метод множителей Лагранжа** (при достаточном времени вычислений).
