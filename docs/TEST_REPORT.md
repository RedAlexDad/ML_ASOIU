### 4. Метод барьерных функций
```
x* = [0.55000000, 0.55000000, 0.55000000]
f(x*) = 119.1868750000
Итераций: 8
Допустимо: True
```
**❌ НЕ РАБОТАЕТ** - метод застревает в начальной точке

**Причины:**
1. Градиент функции Розенброка очень большой: ∇f ≈ [-83, -9, 74]
2. Барьер не может "пересилить" такой градиент
3. Метод застревает в локальном минимуме

**Примечание:** Метод барьерных функций теоретически работает ТОЛЬКО когда:
- Оптимум лежит **внутри** допустимой области (не на границе)
- Градиент целевой функции небольшой
- Начальная точка выбрана близко к оптимуму

Для варианта 2 (a=150) метод **НЕ ПРИМЕНИМ**.

## 🔍 Анализ проблем

### Почему метод барьерных функций не работает?

1. **Принцип метода:** Барьерная функция $P(x,r^k) = -r^k \sum \ln[-g_j(x)]$ стремится к бесконечности при приближении к границе.

2. **Проблема:** При $r^k \to 0$ метод приближается к границе, но никогда не достигает её.

3. **Результат:** f(x*) = 119.19 вместо 100.00
