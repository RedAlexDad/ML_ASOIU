## 2. ПОСТАНОВКА ЗАДАЧИ

Требуется найти условный минимум функции при ограничениях:

$$f(x) \to \min, \quad g_j(x) \le 0, \quad j = 1, \dots, m$$

**Метод барьерных функций:**

$$F(x, r^k) = f(x) - r^k \sum_{j=1}^{m} \ln[-g_j(x)] \to \min$$

где $r^k \to +0$ - параметр барьера.

## 3. ПРОБЛЕМА С ФУНКЦИЕЙ РОЗЕНБРОКА

### 3.1. Вариант 2 (оригинальный)

**Функция:**
$$f(x) = \sum_{i=1}^{n-1} \left[ 150(x_i^2 - x_{i+1})^2 + 2(x_i - 1)^2 \right] + 100$$

**Ограничения:**
$$\begin{aligned}
g_1(x) &= x_1^2 + x_2^2 - x_3 - 1 \le 0 \\
g_2(x) &= -x_1 \le 0 \\
g_3(x) &= -x_2 \le 0 \\
g_4(x) &= -x_3 \le 0
\end{aligned}$$

**Известный оптимум:** $x^* = (1, 1, 1)$, $f(x^*) = 100$

### 3.3. Анализ проблемы

**Причина 1: Оптимум на границе**

В точке $x^* = (1, 1, 1)$:
$$g_1(1,1,1) = 1^2 + 1^2 - 1 - 1 = 0$$

Оптимум лежит **НА ГРАНИЦЕ** допустимой области. Метод барьерных функций **не может достичь границы**, так как барьерная функция стремится к бесконечности при $g_j(x) \to 0$.

**Причина 2: Огромный градиент**

В точке (0.55, 0.55, 0.55):
$$\nabla f \approx [-83, -9, 74]$$

Градиент функции Розенброка с $a=150$ **очень большой**. Барьер не может "пересилить" этот градиент.

**Причина 3: Застревание в локальном минимуме**

Метод застревает в точке (0.55, 0.55, 0.55), которая является локальным минимумом вспомогательной функции $F(x, r^k)$.

## 5. ОБЩИЕ ВЫВОДЫ

### 5.1. Когда метод барьерных функций РАБОТАЕТ:

| Условие | Статус |
|---------|--------|
| ✅ Оптимум **ВНУТРИ** допустимой области | РАБОТАЕТ |
| ✅ Градиент функции **небольшой** | РАБОТАЕТ |
| ✅ Функция **гладкая и выпуклая** | РАБОТАЕТ |
| ✅ Начальная точка **близко к оптимуму** | РАБОТАЕТ |

**Пример:** Квадратичная функция с оптимумом внутри куба [0,1]³

**Точность:** до 10⁻¹²

---

### 5.2. Когда метод барьерных функций НЕ РАБОТАЕТ:

| Условие | Статус |
|---------|--------|
| ❌ Оптимум **НА ГРАНИЦЕ** допустимой области | НЕ РАБОТАЕТ |
| ❌ Градиент функции **очень большой** | НЕ РАБОТАЕТ |
| ❌ Функция **сильно нелинейная** | НЕ РАБОТАЕТ |
| ❌ Начальная точка **далеко от оптимума** | НЕ РАБОТАЕТ |

**Примеры:**
- Функция Розенброка (a=150) - оптимум на границе + большой градиент
- Экспоненциальная функция - оптимум на границе

**Погрешность:** до 10¹ (метод застревает)

### 6.1. Рекомендуемые методы

| Метод | Точность | Скорость | Допустимость | Рекомендация |
|-------|----------|----------|--------------|--------------|
| **Множителей Лагранжа** | 10⁻⁹ | Средняя | ✅ | **ЛУЧШИЙ** |
| **Штрафных функций** | 10⁻⁹ | Высокая | ❌ (снаружи) | ХОРОШИЙ |
| **Проекции градиента** | 10⁻² | Низкая | ✅ | СРЕДНИЙ |
| **Барьерных функций** | - | Высокая | ✅ | **НЕ ДЛЯ ВСЕХ ЗАДАЧ** |

### 6.2. Когда использовать метод барьерных функций

**ИСПОЛЬЗОВАТЬ если:**
- Оптимум заведомо внутри допустимой области
- Функция квадратичная или близка к квадратичной
- Градиенты небольшие
- Требуется гарантированная допустимость всех итераций

**НЕ ИСПОЛЬЗОВАТЬ если:**
- Оптимум может быть на границе
- Функция сильно нелинейная (Розенброка, экспоненциальная)
- Градиенты большие (a > 100 в функции Розенброка)

### 6.3. Альтернатива

**Метод множителей Лагранжа** работает **ВСЕГДА** лучше:
- Сходится к точному оптимуму (10⁻⁹)
- Работает когда оптимум на границе
- Не требует выбора начальной точки внутри области

## 7. КОД ДЛЯ ПРОВЕРКИ

### 7.1. Тест с квадратичной функцией (РАБОТАЕТ)

```python
from constrained_optimization import barrier_method
import numpy as np

def quadratic_f(x):
    return (x[0]-0.5)**2 + (x[1]-0.5)**2 + (x[2]-0.5)**2

def quadratic_grad(x):
    return np.array([2*(x[0]-0.5), 2*(x[1]-0.5), 2*(x[2]-0.5)])

constraints = [
    lambda x: x[0] - 1, lambda x: x[1] - 1, lambda x: x[2] - 1,
    lambda x: -x[0], lambda x: -x[1], lambda x: -x[2]
]

result = barrier_method(quadratic_f, quadratic_grad, 
                        np.array([0.2, 0.2, 0.2]), constraints,
                        r0=1.0, C=10.0, eps=1e-8, max_outer_iter=100)

print(f"x* = {result['x_star']}")
print(f"f(x*) = {result['f_star']:.10e}")
# Результат: f(x*) ≈ 2.54×10⁻¹² ✅
```
