# Декомпозиционный план выполнения ДЗ

## Этап 1: Выбор задачи

- [ ] Выбрать одну из задач МО для исследования
  - [ ] Определить область (классификация, сегментация, детекция и т.д.)
  - [ ] Найти описание задачи со ссылками на статьи
  - [ ] Найти репозитории с исходным кодом авторов
  - [ ] Утвердить выбранную задачу

## Этап 2: Теоретическая часть

- [ ] Проработать минимум 2 статьи по выбранной задаче
  - [ ] Статья 1: прочитать и законспектировать
  - [ ] Статья 2: прочитать и законспектировать
  - [ ] Дополнительные статьи (опционально)
- [ ] Описать общие подходы к решению задачи
- [ ] Описать конкретные топологии нейросетей / моделей
  - [ ] Архитектура сети
  - [ ] Особенности архитектуры
- [ ] Математическое описание
  - [ ] Формулы и алгоритмы
  - [ ] Особенности обучения моделей
- [ ] Описать наборы данных
  - [ ] Название и источник
  - [ ] Размер и структура
  - [ ] Метрики и бенчмарки
- [ ] Оценка качества
  - [ ] Описать метрики качества
  - [ ] Записать значения метрик из статей
- [ ] Предложения по улучшению

## Этап 3: Практическая часть

### 3.1 Документация программ авторов

- [ ] Получить исходные коды из репозиториев авторов
  - [ ] Клонировать репозитории
  - [ ] Изучить структуру проекта
- [ ] Создать UML диаграммы
  - [ ] Диаграмма классов
  - [ ] Диаграмма компонентов
- [ ] Визуализация топологии нейросети
  - [ ] Схема архитектуры
  - [ ] Описание слоев и связей

### 3.2 Воспроизведение экспериментов

- [ ] Подготовить окружение
  - [ ] Установить зависимости
  - [ ] Проверить версии библиотек
  - [ ] Подготовить данные
- [ ] Запустить эксперименты авторов
  - [ ] Обучение модели
  - [ ] Тестирование модели
- [ ] Вычислить метрики качества
  - [ ] Повторить эксперимент с теми же параметрами
  - [ ] Сравнить результаты с论文
- [ ] Сделать выводы о воспроизводимости

### 3.3 Улучшение решения

- [ ] Проанализировать результаты
  - [ ] Выявить узкие места
  - [ ] Определить возможности для улучшения
- [ ] Реализовать улучшения
  - [ ] Подготовить код
  - [ ] Провести эксперименты
  - [ ] Оценить результаты

## Этап 4: Отчет

- [ ] Создать титульный лист
- [ ] Написать постановку задачи
- [ ] Оформить теоретическую часть
- [ ] Оформить практическую часть
- [ ] Написать выводы
- [ ] Составить список источников
- [ ] Проверить финальный документ

## Этап 5: Финальная проверка

- [ ] Проверить структуру отчета
- [ ] Проверить форматирование
- [ ] Проверить ссылки на источники
- [ ] Проверить код и документацию
- [ ] Сделать коммит финальной версии
